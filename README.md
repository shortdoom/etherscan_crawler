# etherscan_crawler
Scrapy crawler of etherscan.io token pages (with Javascript Splash)

# You need Splash installed and docker image running!

[Scrapy-Splash](https://blog.scrapinghub.com/2015/03/02/handling-javascript-in-scrapy-with-splash)

```sudo docker run -p 5023:5023 -p 8050:8050 -p 8051:8051 scrapinghub/splash```
